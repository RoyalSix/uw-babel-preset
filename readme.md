# @unfoldingWord/babel-preset

> Babel preset for unfoldingWord apps.

## Install

Using npm:

```sh
npm install @unfoldingword/eslint-config --save-dev
```

or using yarn:

```sh
yarn add @unfoldingword/eslint-config --dev
```

## Setup

add to eslint config:

```js
...
  "extends": [
    ...
    "@unfoldingword"
  ],
  ...
```
